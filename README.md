# IRL Mario Kart

A real-world Mario Kart inspired project combining:

- A computer vision RC car
- A Unity AR game
- SPI display animations
- Real-time sound effects
- Embedded systems and wireless communication

The goal is to create a mixed reality racing experience where physical RC cars interact with virtual Mario-Kart-style mechanics.

---

## Features

- Computer vision object tracking using OpenCV
- Unity AR overlays and game logic
- Animated SPI TFT display on the car
- Real-time sound effect triggering
- Mario Kart style powerups:
  - Banana
  - Blue Shell
  - Star Power Up

---

## Tech Stack

- Python
- OpenCV
- Unity
- MicroPython
- Raspberry Pi Pico / Raspberry Pi
- SPI TFT Display
